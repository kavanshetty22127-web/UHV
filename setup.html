Quiz Application Setup

Option 1: Quick Start (No Installation Required)

Open the quiz directly in your browser:

Start Quiz Directly

Note: This option uses your browser's localStorage for data storage. Data will only be available on this device.

Option 2: With Node.js Server (Cross-Device Support)

For full functionality including cross-device support:

  1. Install Node.js from https://nodejs.org/
  2. Double-click on start_server.bat to start the server
  3. Open your browser and go to http://localhost:3000

Benefits: Share the application with other devices on the same network.

Option 3: With Python Server

If you have Python installed:

  1. Double-click on start_quiz.bat to start the server
  2. Open your browser and go to http://localhost:8000
